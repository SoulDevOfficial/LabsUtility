# LabsUtility
<ins>**Note:** Keep in mind that this directory may not contain the latest versions of some files. If the version you want is not here, please contact *Soul</ins> (soulwatcher._)* <ins>for the version you want!</ins>

# Current Commands
> These list may be outdated at any time. I update it when I can.
```
- /ban (Ban member)
- /base64-decode (Decode a bae64 string to text)
- /base64-encode (Encode text to base64 string)
- /burner-note (Temporary "note" that can be copied, viewed, and deleted at any time without delay)
- /calculator (Basic mathematics calculator)
- /define (Define a word from open dictionary APIs)
- /diff (Not Working)
- /dm (Allows owner to DM user in their server. Cooldown of 10 minutes)
- /fact (Gets a random fact from an API)
- /hash 
- jwt-decode
- **/help** (Outdated)
- /hex-decode (Decodes a hex string to text)
- /hex-encode (Encodes text to a hex string)
- /invite (Invite the bot)
- /joke (Gets random joke from open API)
- /json-format (Formats json)
- /json-validate (Validates json)
- /jwt-decode (no verification)
- /kick (Kick a user in the server)
- /length (Get length of text from inputed string)
- /password-strength (Standard password strength rate)
- /ping (Gets latency of the bot)
- /purge (Mass-Deletes messages in a channel)
- /regex-test (Test regex code)
- /say (Make the bot send text)
- /server-banner (Gets server banner)
- /server-icon (Gets server icon)
- /server-info (Gets server info)
- /shard (Gets current shard information if any)
- /support (Link to Discord)
- /time (Get time via open API)
- /timestamp (Generate a unix time stamp
- /urban (Get definitions from a urban dictionary. WARNING: Some definitions are not safe for all ages)
- /url-check (Powered by Google Safe Browsing)
- /url-decode (Decodes a URL)
- /url-encode (Encodes a URL)
- /user-avatar (Gets user icon)
- /user-banner (Gets user banner)
- userinfo
- /uuid
- /weather
- /wrap
```
